# SVG Auto Converter

Автоматический конвертер SVG в PNG с сортировкой файлов.

## Скачать проект
Полная версия программы доступна для скачивания на Google Диске:
[Скачать архив с программой](https://drive.google.com/file/d/18dB_cmd5yXjrNqnDC3o663SncCh3WKvt/view?usp=drive_link)

## Требования
- Windows
- Python 3 (любая версия, добавленная в PATH при установке)
- `resvg.exe` (инструкция ниже)

## Структура папок
svg-auto-converter/

├── convert.bat          <-- запускаешь

├── svg2png.py

├── resvg.exe

├── SVG/                 <-- сюда кладёшь .svg файлы

├── PNG/                 <-- сюда сохраняются готовые PNG

├── SVG_END/             <-- успешно обработанные исходники

│   └── SVG_SAVE/        <-- (опционально) для долговременного хранения нужных SVG

├── SVG_BAD/             <-- файлы с ошибками конвертации

└── readme.txt(содержание такое же как и в этом тексте)

## Использование
1. Помести один или несколько `.svg` файлов в папку `SVG`.
2. Запусти `convert.bat` (например, двойным кликом).
3. Готовые `.png` с теми же именами появятся в папке `PNG`.
4. Исходники автоматически переместятся:
   - Успешные → `SVG_END`
   - С ошибками → `SVG_BAD` (ошибка запишется в `error.log`)

## Примечания
- Скрипт ничего не перезаписывает, просто добавляет PNG с тем же именем.
- Папки создаются автоматически при первом запуске.
- Папка `SVG_SAVE` (внутри `SVG_END`) не участвует в автоматической обработке.
  Можешь вручную копировать туда нужные SVG-файлы, чтобы сохранить их в оригинале.
  Если папка не нужна — просто удали её (скрипт не требует её наличия).
